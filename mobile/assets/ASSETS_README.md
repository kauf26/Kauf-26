# App Assets Guide

Before submitting to the App Store, you need to replace these placeholder files with your actual assets.

## Required Assets

### icon.png (1024x1024 pixels)
- This is your app icon shown on the home screen
- Must be exactly 1024x1024 pixels, PNG format
- No transparency allowed
- No rounded corners (iOS adds them automatically)
- Suggested design: Dark background (#0a0a0f) with blue marketplace icon (#3b82f6)

### splash.png (2048x2048 pixels recommended)
- Shown while app loads
- Center your logo, background will extend to fill screen
- Use same dark theme (#0a0a0f background)
- Keep important content in center 1242x1242 area

### adaptive-icon.png (1024x1024 pixels) - Android only
- Similar to icon.png but can have transparency
- Content should be in center 72% of the image

## Quick Design Tips

1. Use a simple, recognizable icon (shopping bag, marketplace, or store icon)
2. Keep text minimal or avoid it entirely
3. Test at small sizes (29x29, 60x60) to ensure visibility
4. Use contrasting colors that work on both light and dark backgrounds

## Free Tools to Create Icons

- Figma (figma.com) - Free design tool
- Canva (canva.com) - Easy icon templates
- IconKitchen (icon.kitchen) - Generate all sizes automatically
